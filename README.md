# Full Stack Task Manager

A full stack task management application built with:

* React
* Vite
* Node.js
* Express
* MongoDB Atlas
* Render
* Vercel

## Features

* Create tasks
* Complete tasks
* Delete tasks
* Filter tasks
* Persistent storage using MongoDB
